# ğŸ§¾ HR & Payroll Management System

A full-featured HR and Payroll solution for managing employees, attendance, salaries, leaves, performance ratings, and more.

---

## ğŸ” Features

- Employee registration, attendance, leave management, and check-in/out  
- Department-wise role management (Admin, Dept Head, Accounts, Super Admin)  
- Salary calculation including overtime, bonus, allowance, and leave penalties  
- Freeze/reactivate employee accounts with view-only access  
- 1-to-1 messaging (auto-deletion after 10 days)  
- Employee rating system for performance review  
- Secure login with JWT authentication and role-based access control
